# 🌆 Comparador de Cidades em C

Este projeto em linguagem C permite comparar duas cidades com base em dados econômicos e demográficos, como população, PIB, área e pontos turísticos.

O objetivo é praticar entrada de dados, operações matemáticas e lógica de comparação.

---

## 📌 Funcionalidades

- Entrada de dados de duas cidades
- Cálculo de:
  - 📊 Densidade populacional
  - 💰 PIB per capita
- Comparação entre cidades:
  - Maior densidade
  - Maior PIB per capita
- Exibição do “resultado vencedor” em cada categoria

---

## 🧠 Conceitos utilizados

- Variáveis e tipos de dados (`int`, `long int`, `char`)
- Entrada de dados com `scanf`
- Saída de dados com `printf`
- Operações matemáticas básicas
- Comparações lógicas (`>`)
- Conversão de tipos (casting)

---

## 🏙️ Dados solicitados

Para cada cidade, o programa pede:

- Nome da cidade
- População
- PIB
- Área territorial
- Número de pontos turísticos

---

## 📐 Cálculos realizados

### Densidade populacional

Densidade = População / Área
PIB per capita
PIB Per Capita = PIB / População
⚖️ Comparação entre cidades

O programa compara:

📍 Qual cidade tem maior densidade populacional
💰 Qual cidade tem maior PIB per capita

E exibe o “vencedor” em cada categoria.
